Pandas Dataframes¶

What is a dataframe?¶

A dataframe is a data structure which stores data in rows and columns. Data analysis focused software such as R, SAS, and SPSS, use dataframes to store data in a row and column format. The most basic data frame stores a variable in each column, and the rows represent the number of observations, one row per observation. There are other ways to store data and reperesent data in dataframes which will later explore. Dataframes can be flat, a single standalone table, or relational, multiple tables related/joined by common identifiers.¶

Python language has been increasingly used in data mining as it has many data-centric packages such as Numpy, Scipy, Matplotlib, and Pandas, Graphlab, Cubes... which facilitate data manipulation, analyses, and visualization. Python combined with Jupyter Notebook provide a very poweful data analysis platform. In terms of computing speed and options availability the pandas package is a joy to work with.¶

Pandas which stands for "Python Data Analysis Library" is a Python library which which reads, converts, analyzes, and stores data in the Python dataframe object. Pandas analytic and visualization functionalities are built on the Numpy and Matplotlib libraries respectively. These two packages must be installed for Pandas to run successfully. Both of these poackages are included in your Anaconda installation.¶

Data from diffferent sources or same source but different tables can be collated, merged, and combined to construct dataframes which we will use analytic sets.¶

Another useful statistical method is the correlation between tow numeric variables, which is basically the amount that two variables increase or decrease in parallel to each other. That is if everytime temperature increased by a single degree humidity increases by a factor of 2 we can say that the two numeric variables are perfectly correlated, and their correlation would be one. If this only ahppens 50% of the time then their correlation would be 0.5, if everytime temp increased by 1 the humidity decreased by a factor of 2 then we would say that they are perfectly negatively correlated and their correlation would be -1. A correlation ranges from -1 to 1.¶
